# MindMaking Conversation Dashboard Alpha Feedback

Welcome to the alpha testing program. This repository is for reporting issues, requesting features, and discussing your experience with the platform.

## What we're testing

We're validating that our core platform interface works reliably with your conversational data sources. This alpha focuses on:
- Data integration (email, call recordings)
- vCon conversion and processing
- Platform UI and basic analytics
- Overall stability and performance

## How to participate

### Reporting bugs
If something's broken or not working as expected, [create a bug report](../../issues/new?template=bug_report.md).

Please include:
- Timestamp of when it occurred
- Specific error messages or symptoms
- Steps to reproduce
- How urgently this is blocking you

### Requesting features
Have ideas for improvements? [Submit a feature request](../../issues/new?template=feature_request.md).

### Integration problems
Having trouble connecting your data sources? [Report an integration issue](../../issues/new?template=integration_problem.md).

### General discussion
Use [Discussions](../../discussions) for questions, sharing workarounds, or general feedback that doesn't fit into a specific issue.

## Response times

We'll respond to:
- **Critical issues** (blocking you completely): within 24 hours
- **All other issues**: within 48 hours
- **Discussions**: as time allows
